# RouteFlow (루트플로우)

RouteFlow는 여행 중 '매일의 이동 경로'를 꼼꼼하게 계획하고 재사용할 수 있는 개인 여행 플래너 앱입니다. 
일본 여행처럼 교통수단(기차, 버스, 지하철) 환승과 정확한 승강장 정보가 중요한 여행을 위해 설계되었습니다.

## 주요 기능

*   **대시보드**: 내 여행 일정 목록과 기간을 한눈에 관리
*   **Day & 타임라인**: 일자별로 탭을 나누어 타임라인 형태로 이동 경로 편집
*   **드래그 앤 드롭**: 일정을 쉽게 드래그하여 순서 변경
*   **경로 상세 정보**: 출발지, 도착지, 요금, 소요 시간, 버스/노선 번호, 승강장 게이트, 구글 맵 링크 등 세밀한 데이터 입력
*   **통계 패널**: 각 Day별 총 이동 시간, 거리, 비용, 환승 횟수를 자동 계산
*   **라이브러리 (즐겨찾기)**: 자주 쓰는 경로를 저장하고 다음 여행에서 빠르게 검색하여 참고
*   **공유 및 인쇄**: 일정을 모바일 기본 공유(navigator.share)로 친구에게 전송하거나 깔끔한 인쇄 모드로 PDF 저장

## 로컬 실행 및 개발

이 앱은 React와 Vite를 기반으로 하며 백엔드 없이 `localStorage`에 데이터를 안전하게 저장하는 정적 앱입니다.

1.  패키지 설치: `npm install` (최상위 폴더에서)
2.  개발 서버 실행: `npm run dev`
3.  브라우저에서 `http://localhost:5173` 등 주어진 주소로 접속

## 배포 (GitHub Pages)

정적 호스팅 환경에 쉽게 배포할 수 있습니다.
1.  `npm run build` 스크립트를 실행하면 `dist` 폴더에 배포 가능한 정적 파일이 생성됩니다.
2.  해당 파일을 GitHub Pages, Vercel, Netlify 등에 업로드하여 즉시 서비스할 수 있습니다.
